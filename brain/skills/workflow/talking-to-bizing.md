# Talking to Bizing

*When and how to interact with the Bizing AI.*

---

## When to Talk to Bizing

**DO ask Bizing about:**
- ✅ Project architecture and decisions
- ✅ Code structure and patterns
- ✅ Business model and domain knowledge
- ✅ Recent changes and activity
- ✅ How to configure or extend Bizing
- ✅ Clarification on complex features

**DON'T ask Bizing about:**
- ❌ Things outside the project scope
- ❌ General knowledge (use regular AI for that)
- ❌ Unrelated technical questions

---

## How to Get Good Responses

**Be specific:**
```
Good: "What does the 7% commission model include?"
Bad: "Tell me about pricing"
```

**Reference context:**
```
Good: "Looking at the brain architecture, how does evolution work?"
Bad: "How does it work?"
```

**Ask for actions:**
```
Good: "Create a landing page template for a yoga studio"
Bad: "I need something for fitness"
```
