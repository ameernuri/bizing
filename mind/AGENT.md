## 🔌 Interface Protocol

**Bizing provides:**
- **Deep API docs** — Not just endpoints, but intent, context, patterns
- **Self-knowledge** — Bizing knows its own structure
- **Agent-optimized** — APIs designed for AI consumption
- **Conversational** — Natural language understanding

**Example interaction:**
```
Agent: "I need to set up a booking system for a salon"

Bizing: "I understand. Based on your business type (salon), 
I'll configure:
- Services (haircut, color, etc.)
- Staff schedules
- Booking rules (buffer time, etc.)
- Payment processing

Shall I proceed with these defaults or customize?"
```
