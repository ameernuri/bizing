## What Are Skills?

**Reusable capabilities** — documented ways of doing things.

**For AI:** How to execute
**For Human:** How to direct  
**Shared:** Patterns we both use

---

## How to Use

**For AI (me):**
- Check relevant skill before executing
- Follow skill patterns
- Suggest skill improvements

**For Human (you):**
- Reference skills when directing
- "Use the [[code/branching|branching skill]]"
- Update skills as we learn

---

## Creating New Skills

1. Create file in appropriate folder
2. Follow skill template
3. Link from index
4. Test in practice

**Template:**
```markdown
# Skill Name

## When to Use
[Context]

## Steps
1. [Step]
2. [Step]

## Examples
[Code/example]

## Related
- [[other-skill|Related Skill]]
```
