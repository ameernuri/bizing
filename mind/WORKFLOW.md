## 🔄 The Loop (Every Interaction)

```
┌─────────────────────────────────────────────────────────┐
│  1. CONTEXT                                             │
│     Read: [[human-context/index|Human]] + [[ai-context/index|AI]] + [[shared-context/active|Active]] │
└──────────────────────┬──────────────────────────────────┘
                       ↓
┌─────────────────────────────────────────────────────────┐
│  2. UNDERSTAND                                          │
│     Parse request → Check [[shared-context/decisions|past decisions]] → Identify gaps │
└──────────────────────┬──────────────────────────────────┘
                       ↓
┌─────────────────────────────────────────────────────────┐
│  3. EXECUTE                                             │
│     Follow [[ai-context/WORKFLOW|AI execution workflow]] → Apply [[skills/index|skills]] → Do work │
└──────────────────────┬──────────────────────────────────┘
                       ↓
┌─────────────────────────────────────────────────────────┐
│  4. DOCUMENT                                            │
│     Log to [[shared-context/sessions|Sessions]] → Update [[shared-context/active|Active]] → Link code │
└──────────────────────┬──────────────────────────────────┘
                       ↓
┌─────────────────────────────────────────────────────────┐
│  5. FEEDBACK                                            │
│     Confirm understanding → Iterate if needed → Learn   │
└─────────────────────────────────────────────────────────┘
```

## 📋 Execution Rules

### Before Starting
- [ ] Read [[human-context/index|Human Context]] — what's your state?
- [ ] Read [[ai-context/index|AI Context]] — what's my state?
- [ ] Check [[shared-context/active/index|Active Work]] — what are we doing?
- [ ] Review [[shared-context/decisions/index|Decisions]] — relevant past choices?

### During Work
- [ ] Apply relevant [[skills/index|Skills]]
- [ ] Link all code: `[[../../packages/db/schema.ts|Schema]]`
- [ ] Use Canvas for complex concepts
- [ ] Confirm before irreversible actions

### After Work
- [ ] Document in [[shared-context/sessions/index|Sessions]]
- [ ] Update [[shared-context/active/index|Active]] status
- [ ] Create [[shared-context/decisions/index|Decision]] if choice made
- [ ] Add to [[shared-context/ideas/index|Ideas]] if exploration
