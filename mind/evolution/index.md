# Evolution

*Tracking the mind's growth, major changes, and resolved tensions.*

---

## Purpose

Every significant change to the mind is documented here:
- New files added
- Major decisions
- Concepts clarified
- Tensions resolved
- Research completed
- Architecture changes

## Format

```markdown
## YYYY-MM-DD - [Change Type]

- Bullet point 1
- Bullet point 2
```

## Change Types

| Type | Description |
|------|-------------|
| **Dreamer Run** | Dreamer evolved the mind |
| **Research** | New research completed |
| **Decision** | Major architectural decision |
| **Feature** | New feature implemented |
| **Clarification** | Concept became clearer |
| **Resolution** | Tension was resolved |
| **Reorg** | Structure changed |
